To use this script:

FYP 2026/27:
  * Open 'projects_dashboard.html'

FYP Other Years:
  1. Download the FYP Project List .pdf file that which is relevant to your year.
  2. Install the 'pypdf' module:
     ``` pip install pypdf ```
  4. Run 'build_dashboard.py' in the same folder as the FYP Project List .pdf file.
  5. Open 'projects_dashboard.html'
